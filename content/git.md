## Clone

### Simple Clone
```bash
git clone <repo_url>
```

### Clone with Submodules
```bash
git clone --recursive <repository_url>
```

## Switch Branch
```bash
git checkout <branch> 
```
or 
```bash
git switch <branch>
```


## Check Status
```
git status
```


## Pull
```
git pull
```

## Push

```bash
git add .
git commit -m "update"
git push origin main
```
